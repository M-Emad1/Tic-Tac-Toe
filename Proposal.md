# Project Proposal: Tic Tac Toe with AI Opponent

## Description
A classic Tic Tac Toe game featuring an AI opponent that utilizes the Minimax algorithm to provide a challenging and strategic experience.

## Agent Type
* **Type:** Goal-based Agent

## PEAS Analysis
* **Performance:** Winning or tying the game and minimizing moves.
* **Environment:** 3x3 Grid.
* **Actuator:** Placing X or O in an empty cell.
* **Sensor:** Digital Board State.

## ODESA Analysis
* **Observability:** Fully-observable
* **Determinism:** Deterministic
* **Episodic:** 
* **Static:** Static 
* **Agents:** Single-Agent
